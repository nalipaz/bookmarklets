Bookmarklets by Nicholas Alipaz

Below are various bookmarklets that I have written to perform everyday tasks. To use these just click and hold on the link and drag it to your bookmarks bar. Then whenever you need them just click the link in your address bar.

Lowercase
Changes selected text in input or textarea to lowercase.
Title Case
Changes selected text in input or text area to Title Case.
Uppercase
Changes selected text in input or text area to UPPERCASE.

The source code for these bookmarklets can be found in my GitHub repository.
